Meal Planner - Plan4Meals

Meal Planner - Plan4Meals is an Android application focused on meal planning and food organization. Features include calendar-based scheduling, recipe management, shopping list generation, meal prep tracking, multilingual support and AI-powered tools for recipe import and weekly planning.

  • Recipe management

Palavra

Palavra is a browser-based word game inspired by Wordle and adapted for Portuguese-speaking players. Developed using JavaScript, HTML and CSS, the project focuses on game logic, state management and responsive user interface design. Players can choose between multiple difficulty levels and enjoy unlimited play sessions. Future improvements include category-based challenges and additional gameplay mechanics.

Features

  • Portuguese word database
  • Multiple difficulty levels
  • Visual feedback system

Technologies

  • HTML
  • CSS
